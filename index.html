Container Buttons

When enabled, the HoverText of Container Buttons like Container Controller Button, Inventory Container Buttons,... will be only one line, no longer have KeyTip descriptions below. But it will be very convenient for people who already know how to use those buttons!

Hud Screen

Vanilla: Off. This feature will make the Hud always show in the background instead of hiding. Be careful before enabling it because it will corrupt the Touch Control.

Vanilla: Off. This setting will cause the Item Name Text to be one line only, even if the item is enchanted. should not be enabled because its Background width may not be correct.

Loading Screen

Vanilla: On. If this is off it will remove the Background behind the Loading Screen and leave what you see;-;

Vanilla: Off. When enabled, the Loading Bar is always visible regardless of the type of Loading Screen.
